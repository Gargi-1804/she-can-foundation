# she-can-foundation
# She Can Foundation — Frontend Internship Task

A simple webpage built for **She Can Foundation** as part of the Frontend Development Internship selection task.

---

## About the Project

I built this webpage as my internship submission for She Can Foundation. The task was to create a basic single-page website for the NGO using HTML and CSS. I tried to make it look clean and meaningful — something that actually represents what the foundation stands for.

She Can Foundation works towards empowering women through education, skill development, and community support. I wanted the design to feel warm, hopeful, and professional at the same time.

---

## What's Included

- NGO name and branding
- Hero section with a main heading
- About section explaining the foundation's mission
- An image representing women empowerment
- A "Get Involved" button
- Footer with basic links

---

## Extra Features I Added

Since I wanted to go beyond the basics, I added a few more things:

- **Dark Mode** — toggle button in the navbar to switch between light and dark theme
- **Scroll Animations** — sections and cards animate in as you scroll down
- **Responsive Design** — works on mobile, tablet, and desktop
- **Animated Hero Section** — a soft blob animation in the background
- **Custom Cursor** — subtle interactive cursor effect
- **Hover Effects** — buttons and cards respond to mouse interactions
- **Stats Section** — shows the foundation's impact numbers

---

## Tech Used

- HTML5
- CSS3 (animations, CSS variables, flexbox, grid)
- Vanilla JavaScript (dark mode, scroll reveal, cursor)
- Google Fonts (Playfair Display + DM Sans)

---

## How to Run

Just download the `she-can-foundation.html` file and open it in any browser. No installations, no dependencies — it works right away.
